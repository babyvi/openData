<Strong>基于Python的政府开放数据爬虫</Strong>


*   spider为爬虫框架
    *   Spider对传入的URL进行爬取<br>
        对爬取的数据使用JSonParse进行解析<br>
        内置写入Excel与xsv函数
    *   JSonParse   对爬取得json数据进行解析
    *   Writer  将数据写入Excel表中

*   charts为可视化模块
*   fsSpider 与 gzSpider 分别是相应的爬虫文件
*   source为数据保存目录
